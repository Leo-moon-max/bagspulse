# BagsPulse

BagsPulse is a simple Token Health Score dashboard built for the Bags ecosystem.

## Problem
Token creators launch blindly without understanding liquidity, volume, or risk concentration.

## Solution
BagsPulse analyzes a token using public API data and generates a health score instantly.

## Features
- Live token data fetch
- Liquidity scoring
- Volume scoring
- Risk warning system
- Simple boost suggestion logic

## Tech Stack
- HTML
- CSS
- JavaScript
- Dexscreener Public API

## Future Vision
- Whale tracking
- Boost ROI analyzer
- AI strategy assistant
